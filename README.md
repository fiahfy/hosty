# Hosty

This is an Electron application for hosts file management.

### Setup
```
$ git clone https://github.com/fiahfy/hosty.git
$ cd hosty
$ npm i
```

### Run & watch renderer
```
$ npm run start
```

### Run & watch renderer(HMR)
```
$ npm run start:hot
```

### Packaging App
```
$ npm run package
```
